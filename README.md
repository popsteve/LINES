# Mini Metro 克隆项目

这是一个使用Godot引擎和GDScript开发的Mini Metro风格游戏。Mini Metro是一款极简主义交通线路管理游戏，玩家需要绘制线路连接各个车站，并管理乘客流量。

## 游戏特性

- 极简主义设计风格
- 基于节点的交通网络系统
- 动态乘客生成和流动
- 资源管理（有限的线路和车辆）
- 随时间增长的游戏难度
- 过载站点管理机制

## 项目结构

```
MiniMetroClone/
├── assets/        # 游戏资源（图像、音效等）
├── scenes/        # Godot场景文件
└── scripts/       # GDScript脚本文件
```

## 如何运行

1. 安装Godot引擎（建议4.x版本）
2. 克隆此仓库
3. 在Godot中打开项目
4. 点击"运行"按钮开始游戏

## 游戏控制

- 鼠标左键拖拽：创建/修改线路
- 右键点击：删除线路段
- 空格键：暂停/继续游戏
- 更多控制将在开发过程中添加
